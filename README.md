# MathU

***

## structs
### Vector3
- Usage: Vector3 vector = new Vector3(x, y, z);
- Vector3.Magnitude()
- Vector3.Normalized()
- Vector3.ToString()
- Operations: *, +, /, -

### Matrix
- Usage: Matrix matrix1x3 = new Matrix(1, 3);
- Operations: *
- Features: Will throw exception when multiplication not supported.
